Current Studies: Programming & Web Development

Programming

The programming unit covers the basics of programming in Java. The first part of the unit is taught with processing this is a program that allows users to code simple graphics onto an interface. Then from there its onto using an IDE (in this case eclipse) to write console applications, again in Java. The course covers many topics an overview of which is provided in the list below:

  • Software Life Cycle and Design Methodology
  • Verification and testing
  • Application of standards and conventions
  • Constructs and features of high level programming languages
    • Abstraction
    • Operators
    • Use of Libraries
    • Simple I/O
  • Data Types
  • Object orientated design
These Concepts arent always taught separately most times they are taught and used togther e.g. once you have learnt programming conventions you will use them throught the other sections, aswell as throught the course and possibly beyond. The image above it an example of a program built using processing, its useful when learning programming as its visual and often errors made in the code can be visual seen unlike a console program.

Web Development

This unit provides an introduction to modern web design and development. This means that it covers both web design which is the visuals of a web page (HTML and CSS) and web development which is the functionality of a web pages and makes use of technologys such as JavaScript,PHP and MySQL. Some of the topics this course includes are included below:

  • Introduction to the client-server model
  • The use of high-level tools for design and development
  • The production of HTML5 video
  • The DOM element Model
This unit is marked through projects, this website being one of them. This website is used to assess my skills in HTML and CSS, which was covered in the first 7 weeks of the course. I will also create a website which contains a video but this hasn't been implmented as this is part of the next assignment which is control of HTML 5 video through JavaScript. JavaScript is taught up intill the christmas break after this the unit covers the use of PHP and MySQL for the backend of websites.
